Calificaciones

Programa sencillo y gratuito para calcular rápidamente la calificación en base al puntaje obtenido de un puntaje máximo.

Características

Inicio

Calcula la calificación obtenida dados los aciertos obtenidos

Solo con escribir los aciertos, sin regla de tres ni cálculos

tablas

Genera tablas con los  aciertos y calificaciones correspondientes

Es posible imprimir la tabla por si se desea trabajar sin el programa

Descargar

Puedes descargar el programa desde GitHub releases
